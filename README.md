# FPGA_Cicero
Integration of the Nir Navon Lab's "DMDCicero" (Cicero Word Generator + DMD capabilities) with the Ariel Sommer Lab's FPGA script for the OpalKelly XEM7001.


## Device Dependencies
- .NET 3.5 Framework
- NI DAQ
- NI VISA
- NI ...
- OpalKelly XEM7001
